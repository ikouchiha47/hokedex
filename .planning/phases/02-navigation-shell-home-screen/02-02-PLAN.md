--- GalleryBottomSheet.tsx ---

Create src/components/GalleryBottomSheet.tsx.

This is a custom bottom sheet using react-native's PanResponder and react-native-reanimated v4.

Key imports:
```ts
import Animated, { useSharedValue, useAnimatedStyle, withSpring } from 'react-native-reanimated';
import { PanResponder, Dimensions } from 'react-native';
```
Do NOT import useAnimatedGestureHandler — it does not exist in Reanimated v4.

Props:
```ts
interface GalleryBottomSheetProps {
  isOpen: boolean;
  onClose: () => void;
}
```

Constants:
```ts
const SCREEN_HEIGHT = Dimensions.get('window').height;
const SHEET_HEIGHT = SCREEN_HEIGHT * 0.6;   // 60% of screen
const CLOSED_Y = SHEET_HEIGHT;              // off-screen (translated down)
const OPEN_Y = 0;                           // fully visible
```

translateY is a useSharedValue<number> initialized to CLOSED_Y.

Effect: when isOpen changes, animate translateY:
```ts
useEffect(() => {
  translateY.value = withSpring(isOpen ? OPEN_Y : CLOSED_Y, { damping: 20, stiffness: 200 });
}, [isOpen]);
```

PanResponder handles swipe-down to close:
```ts
const panResponder = PanResponder.create({
  onMoveShouldSetPanResponder: (_, gs) => gs.dy > 8,
  onPanResponderMove: (_, gs) => {
    if (gs.dy > 0) translateY.value = gs.dy;
  },
  onPanResponderRelease: (_, gs) => {
    if (gs.dy > 80) {
      translateY.value = withSpring(CLOSED_Y, { damping: 20 });
      onClose();
    } else {
      translateY.value = withSpring(OPEN_Y, { damping: 20 });
    }
  },
});
```

Animated style:
```ts
const animatedStyle = useAnimatedStyle(() => ({
  transform: [{ translateY: translateY.value }],
}));
```

Render an Animated.View with:
- position: 'absolute', bottom: 0, left: 0, right: 0, height: SHEET_HEIGHT
- backgroundColor: '#0e0f23', borderTopLeftRadius: 16, borderTopRightRadius: 16
- Apply animatedStyle and {...panResponder.panHandlers}
- Inside: drag handle (32×4 rounded pill, backgroundColor 'rgba(255,255,255,0.2)', centred, marginTop 12), then GalleryPivot.

Export as named export.
