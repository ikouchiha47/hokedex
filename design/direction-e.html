C · Palette & colour semantics
Primary · auroraviolet→magenta
Active tab, primary CTAs, pivot underlines, L2/L3 card accents, FAB, profile ring. The brand gradient.
Info · cyan#22d3ee
Map tiles, place chips, local-mode glow, note-card borders, "transcribed" tag. Anything cartographic or informational.
Planned · amber#fbbf24
Planned events, calendar event dots, countdowns, weather sun, "en route" state, L4 featured tier.
Streaks · emerald#34d399
High face-confidence, "here" state on event show, place likes, recurring-event dots, healthy status.
Voice · rose#f43f5e
Voice cards + recording UI, drift alerts, delete actions, danger outlines, "no model" warnings.
Surface · voiddark base
#0a0a12 base with violet/magenta radial glows top-corners. Surfaces #11111c / #161624. Keeps it dark but never flat black.
